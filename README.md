# Todos UI

A React + TypeScript frontend for the Spring Boot **Todos API**.

This UI integrates with:

- Categories CRUD
- Todos CRUD + archive (soft delete)
- Filtering todos by category
- Sorting todos by `createdAt`, `title`, or `completed` with `ASC` / `DESC`

## Tech Stack

- React (Vite)
- TypeScript
- React Router
- Tailwind CSS
- Vitest + Testing Library

## Setup

1. Install dependencies

```bash
npm install
```
